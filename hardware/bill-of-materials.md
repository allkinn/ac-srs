# Bill of Materials (BOM)

## Required Components
- Arduino Nano CH340G (ATmega328P)
- USB Cable Mini
- 2x Laser Module KY-008 5V
- 2x LDR Module LM393
- AHT20 Temperature & Humidity Sensor (I2C)
- Active Buzzer
- PCB
- Jumper Wires
- Adaptor 5V 2A
- UTP Cable Cat5e 15m


## Optional Components
- DC Female Jack
- Screw Terminal Block KF350
